What is Stormly?

Stormly is an app designed to solve low-quality weather measurements in smaller areas. We decided to create a device with a connection to a personal database, which stores weather information. This data can be easily accessed by logging into the website. Each user has a unique account with weather information from his device.

Our idea for this station includes two main segments. The first one is a hardware part that contains a microcontroller with all necessary measuring and communication equipment. The crucial sensors are temperature, pressure, and humidity. The second part is software such as connectivity with database, and web service.

Our goal

The majority of online services don’t offer accurate forecast updates for small towns. To solve this problem we came up with the idea of an online weather service where we can store gathered data and display the most recent ones. The collected registry can be used for climate change approximation compared with different intervals.

The weather information website’s main purpose is to collect, compute and display precise weather data via web service. Measurements conducted on a wider range require using multiple weather centers. Therefore, the only and the right approach based on budget assumption and technical complexity is microscale.

Equipment

Arduino UNO microcontroller

Arduino UNO is heart of our project. It is a microcontroller board based on a removable, dual-inline-package (DIP) ATmega328 AVR microcontroller. It has 20 digital input/output pins (of which 6 can be used as PWM outputs and 6 can be used as analog inputs).

Along with all sensors, Arduino is responsible for measuring and processing collected data. In addition to that, processed data is next send using built-in serial port.
